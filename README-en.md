## 📋 Course Descriptions

### 📘 01 — Basic Programming `010069` | 3 Credits
> **Prerequisite:** None
>
> This course introduces students to the **fundamental concepts of computer programming**
> using **C++** as the primary instructional language. Topics covered include program structure,
> data types, control flow (conditionals and loops), functions, arrays, pointers, and
> basic file input/output operations. Laboratory sessions are conducted in **Microsoft Visual Studio**.
> This course serves as a prerequisite for *Data Structures & Algorithms* and all
> subsequent programming courses within the MIS curriculum.
>
> **Learning Outcomes:** Upon completion, students will be able to design, implement,
> debug, and document structured programs using C++.

---

### 📗 02 — Databases `010071` | 3 Credits
> **Prerequisite:** Basic Programming
>
> This course provides a comprehensive introduction to **relational database theory and practice**.
> Topics include the relational data model, **Entity-Relationship (ER) modelling**,
> **normalisation (1NF–3NF)**, and structured query language (**SQL** — DDL, DML, DCL).
> Practical sessions involve hands-on implementation using **MySQL** and
> **Microsoft SQL Server**. Students will design, build, and query relational databases
> in the context of real-world business scenarios relevant to the Finance and Marketing sector.
>
> **Learning Outcomes:** Upon completion, students will be able to design normalised
> relational schemas, write complex SQL queries, and implement database solutions
> for enterprise information systems.

---

### 📙 03 — Management Information Systems `010045` | 3 Credits
> **Prerequisite:** Basic Informatics
>
> This course examines the **strategic role of information systems** in modern organisations.
> Core topics include the classification of IS (TPS, MIS, DSS, ERP, CRM, ESS),
> systems analysis and design methodologies (**DFD, UML, ERD**), and the application
> of IT in business decision-making and operational management.
> Case studies draw from Vietnamese enterprises and the Finance-Marketing industry.
> Integrated practical sessions involve group-based systems analysis projects.
>
> **Learning Outcomes:** Upon completion, students will be able to analyse organisational
> information needs, evaluate IS solutions, and apply systems thinking to
> business problem-solving.
